# Syed Safwan Portfolio

Personal portfolio website built with Next.js to showcase projects, skills, certifications, and experience.

## Live Site

- https://syedsafwan24.github.io/portfolio/

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Framer Motion

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open http://localhost:3000

## Production Build

```bash
npm run build
```
